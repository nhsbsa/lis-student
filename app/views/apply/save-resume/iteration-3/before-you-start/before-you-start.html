Before you start

Apply online for help with NHS costs

We will now ask you 7 questions to make sure you can apply online at this time.

If you cannot apply online, you can still apply using the paper form, called an HC1.

How to get an HC1 form

You can get an HC1 form from your:

  • local pharmacy
  • GP practice
  • local Jobcentre
  • Citizens Advice

Or, you can:

  • download an HC1 form (PDF: 452KB) (opens in a new tab) and print it off.
